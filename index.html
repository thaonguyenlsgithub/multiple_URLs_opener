Installation & Usage

Get up and running in under a minute. No configuration needed.

1

Download & Extract

Download the extension from Google Drive and extract the ZIP file.

2

Load into Chrome

Go to chrome://extensions/, enable Developer mode, and click "Load unpacked".

3

Start Extracting

Click the extension icon, paste URLs, and hit "Extract URLs".

FAQ

How do I install this extension? →

Download the extension from Google Drive, extract the ZIP, go to chrome://extensions/, enable Developer mode, click "Load unpacked", and select the folder. That's it!

Can I use this on Firefox or Edge? →

Currently this is a Chrome extension only. Firefox/Edge support could be added in the future if there's demand.

What happens to my URLs? →

All processing happens locally in your browser. No URLs are sent anywhere—complete privacy.

Why does it say "only one tab opens"? →

This might be a browser permission issue. Try enabling "Delay opens" in settings or reloading the extension from chrome://extensions/.

Can I extract URLs from other formats? →

Yes! The extension supports pipe-delimited (|), newline-separated, comma-separated, and mixed text formats.
